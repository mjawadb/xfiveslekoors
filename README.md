# 🔥 X-Five Slekoors

Website official buat X-Five Slekoors. Project ini dibikin full animasi biar UI-nya nggak kaku dan ngebosenin.

## Tech Stack
Dibangun pakai ekosistem React dengan beberapa tambahan library buat styling dan animasi:
- React.js 
- Tailwind CSS
- Framer Motion, Anime.js, AOS, & React Reveal (buat urusan animasi)
- Swiper (buat carousel)

## Cara Jalanin di Lokal
Kalau mau clone dan coba run sendiri:

1. Pastikan udah install Node.js
2. Clone repo ini ke lokal
3. Jalanin `npm install` di terminal buat narik semua dependency
4. Ketik `npm start`
5. Tinggal buka `http://localhost:3000` di browser

## Build & Deploy
Buat nge-build versi production:
```bash
npm run build
```
